# Earthquake Studio - Music Platform

A modern music streaming platform built with Next.js 15.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Audio**: Cloudinary CDN
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Home
- `/music-library` - Full music catalog
- `/about` - About us
- `/contact` - Contact form
- `/faq` - FAQ
- `/privacy` - Privacy policy
- `/terms` - Terms of service

## Environment

No environment variables required for frontend.

## Deployment

Push to GitHub and connect to Vercel for automatic deployments.
